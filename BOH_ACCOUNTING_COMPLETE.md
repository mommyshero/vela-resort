## 🔧 ระบบ Auto-Sync:

### ทำงานอย่างไร:

```javascript
// ทุกครั้งที่เพิ่มรายการ Petty Cash
function addPettyEntry() {
    pettyEntries.push(newEntry);
    localStorage.setItem('vela_petty', JSON.stringify(pettyEntries));
    
    // ✅ Auto-sync to General Ledger
    syncPettyToGeneralLedger(pettyEntries);
    
    alert('✅ บันทึกรายการเรียบร้อยแล้ว! (อัพเดทบัญชีใหญ่แล้ว)');
}
```

### ตัวอย่างการ Sync:

**รายการรายย่อย:**
```json
{
  "id": "PET001",
  "type": "expense",
  "category": "food",
  "description": "ค่าข้าวกล่อง 4 กล่อง",
  "amount": 240,
  "date": "2026-01-01"
}
```

**→ Sync เป็นบัญชีใหญ่:**
```json
{
  "id": "GL001",
  "date": "2026-01-01",
  "account": "5101-001",
  "accountName": "ค่าใช้จ่ายอาหาร",
  "description": "ค่าข้าวกล่อง 4 กล่อง",
  "debit": 240,
  "credit": 0,
  "refId": "PET001",
  "refType": "petty-cash"
}
```
