## 📝 การแก้ไขข้อความ "ฤดูเขียว" → "ฤดูร้อน/ฤดูฝน"

### วิธีที่ 1: แก้ผ่าน WordPress Admin
1. เข้า `https://vela-resort.com/wp-admin`
2. ไปที่ **Pages** (หน้า)
3. เลือกหน้าที่มีข้อความ "ฤดูเขียว"
4. คลิก **Edit**
5. แก้ไขข้อความ:
   - "ฤดูเขียว" → "ฤดูร้อน" (มี.ค.-พ.ค.)
   - "ฤดูเขียว" → "ฤดูฝน" (มิ.ย.-ต.ค.)
6. คลิก **Update**

### วิธีที่ 2: แก้ผ่าน Database (MySQL)
```sql
-- ค้นหาข้อความ "ฤดูเขียว"
SELECT * FROM wp_posts WHERE post_content LIKE '%ฤดูเขียว%';

-- แก้ไขเป็น "ฤดูร้อน"
UPDATE wp_posts 
SET post_content = REPLACE(post_content, 'ฤดูเขียว', 'ฤดูร้อน') 
WHERE post_content LIKE '%ฤดูเขียว%';
```

## 📅 การจัดการระบบจอง

### เปิด/ปิด การจอง

1. เข้า WordPress Admin
2. ไปที่ **Bookings** หรือ **Reservations** (แล้วแต่ plugin ที่ใช้)
3. เลือกวันที่ต้องการ
4. เปลี่ยนสถานะ:
   - ✅ **Available** (เปิดจอง)
   - ❌ **Blocked** (ปิดจอง)
   - 📝 **Reserved** (จองแล้ว)

### Plugin ที่แนะนำสำหรับระบบจอง:
- **WooCommerce Bookings**
- **Booking Calendar**
