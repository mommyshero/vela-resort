⏸️ พร้อมใช้งาน - คลิกปุ่มเริ่ม

🎤 Single Mode

เสียงเดิม 100% ไม่มีการซ้อน เหมาะสำหรับอ้างอิง

👯 Double Mode

เสียงคู่ 50/50 สร้างความหนาแบบธรรมชาติ

🎚️ Thick Mode

เสียงหนา 70/30 เหมาะสำหรับร้องนำ

🌊 Chorus Mode

เอฟเฟกต์คอรัส เสียงกว้าง 立体感

🎼 Octave Mode

ซ้อนเสียงออกเทฟ (สูง/ต่ำ 12 semitones)

📖 วิธีใช้งาน

  • คลิกปุ่ม "เริ่มใช้งาน" และอนุญาตให้เข้าถึงไมโครโฟน
  • เลือกโหมดการซ้อนเสียง (Single, Double, Thick, Chorus, Octave)
  • ปรับค่า Delay, Mix, Detune ตามต้องการ
  • พูดหรือร้องเพลง เพื่อฟังผลลัพธ์แบบเรียลไทม์
  • ใช้หูฟังเพื่อฟังผลลัพธ์ได้ชัดเจนที่สุด

🎯 การประยุกต์ใช้

  • 🎤 นักร้อง: ซ้อนเสียงร้องให้หนาขึ้น
  • 🎙️ พอดแคสต์: เพิ่มความอุ่นให้เสียงพูด
  • 🎸 นักดนตรี: สร้างเสียงคู่ให้เครื่องดนตรี
  • 🎚️ Mixing: ใช้เป็น Reference สำหรับเปรียบเทียบ

💡 เคล็ดลับ

  • ใช้หูฟังเสมอเพื่อป้องกันเสียง Feedback
  • เริ่มจากค่า Default ก่อนแล้วค่อยปรับ
  • โหมด Double เหมาะสำหรับเสียงร้องหลัก
  • โหมด Chorus เหมาะสำหรับเสียงพื้นหลัง
